# Project Apple

A Discord bot written in Rust using the `serenity-rs` library, designed as an open-source alternative to the pay-to-use "bleed" bot.

**This project is currently a work in progress** due to my school and other things. I will do my best to actively work on and improve this project.

## Why? What is this project about?

Many of you may be familiar with the Discord bot named "bleed." You may have wanted to add it to your server but found that its features are locked behind a "pay-to-use" system. To access it, you must join their support server and pay a fixed subscription fee, essentially turning it into a paid service.

I strongly dislike this practice, which is why I’m creating an open-source alternative. My goal is to provide a free and accessible bot for everyone. I hope that no one will take my code and put it behind a "pay-to-use" system.
